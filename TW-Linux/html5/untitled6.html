ls

Lists all items in the current directory.

You must have installed Linux or run it with through a virutal machine

You must have opened the terminal with the combination of Ctrl+Alt+T or from the menu → terminal

ls is short for list and it does as it says. It lists every file in the current directory.

Q: For example I have a directory named “A”, and in it there are files called “a1”,”b1”.

What will be the output from the terminal when I give the input “ls” ?

A: The answer is simple. It depends. Read once again the description for the command pwd and you will now why it depends. So if we are in currently in the location of A it will display a1, b1. Otherwise the answer is: what is having in the directory of {pwd}.
