Cp and Mv

The commands cp and mv are coping and moving files through the file system.

You must have installed Linux or run it with through a virutal machine

You must have opened the terminal with the combination of Ctrl+Alt+T or from the menu â†’ terminal

Cp for copy and mv for move are commands which are used to organize your files in other directories and copy their content.

the syntax is:

cp <file to be copied> <where to be copied>

Mv <file to be moved> <where to be moved> * the mv command can be used also to rename files.
  1. Open Linux Terminal
  2. Make a file, like it is described on the previous chapter
  3. Type the command cp to copy the file
  4. Type as a First parameter the name of the file that you are going to copy
  5. Type as a Second Parameter the location which you want the file to be copied at.
  6. Press Enter
  7. Go to the location which you provided on step 5.
  8. After you list the elements of the current directory, you will find out that there is a copy of the original file.
